# Global Solution IoT - AeroNet Agro

Projeto desenvolvido para a Global Solution 2026/1, utilizando ESP32 no Wokwi para simular uma solução IoT aplicada ao agronegócio.

## Descrição

O projeto monitora a umidade do solo por meio de um potenciômetro, simulando um sensor agrícola. Quando a umidade fica abaixo de 40%, o sistema identifica solo seco e ativa o alerta de irrigação. Também existe um botão para acionamento manual.

## Componentes

- ESP32
- Potenciômetro simulando sensor de umidade do solo
- Botão para irrigação manual
- LED vermelho para alerta
- LED verde para status normal
- Display LCD 16x2 I2C
- WebServer com endpoints JSON

## Funcionalidades

- Leitura da umidade do solo
- Alerta automático para solo seco
- Acionamento manual da irrigação
- Exibição das informações no LCD
- Comunicação Wi-Fi
- Dashboard Web
- Endpoints JSON

## Endpoints

- /dados
- /status
- /irrigacao
